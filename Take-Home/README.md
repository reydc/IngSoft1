## Take-Home (2020)

Consiste en hacer analisis de tres módulos de una posible implementación del juego "Secret Voldemort":

* Módulo de Usuarios
* Módulo de Turno
* Módulo de Partidas

Para cada uno de ellos se debe entregar:

- Alcance del módulo
- Tres casos de uso
- Diagrama de flujo de datos
- Diagrama de clases

Tiempo de entrega: 1 semana y 5 días
